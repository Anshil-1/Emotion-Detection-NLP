# Emotion Detection using NLP

## Project Overview

This project predicts human emotions from text using Natural Language Processing (NLP) and Machine Learning.

The model classifies text into emotions such as:

- Joy
- Sadness
- Anger
- Fear
- Love
- Surprise

## Technologies Used

- Python
- Pandas
- NLTK
- Scikit-Learn
- TF-IDF Vectorization
- Logistic Regression

## Workflow

1. Data Collection
2. Text Preprocessing
3. TF-IDF Vectorization
4. Label Encoding
5. Train-Test Split
6. Model Training
7. Prediction

## Text Preprocessing

- Lowercasing
- Punctuation Removal
- Stopword Removal
- Tokenization

## Model Used

Logistic Regression

## Example

Input:

I am feeling very happy today

Output:

Joy
